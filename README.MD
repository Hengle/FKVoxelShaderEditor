#说明

一个很简单的MagicaVoxel文件加载工具，包括了MonoShader编译，Shader动态编辑等功能
A simple monogame editor, it can load assets from MagicaVoxel, and compile monoshader *.fx files, and can edit your shader code easily like ShaderToy.

#使用

请手动拷贝C:\Program Files (x86)\MSBuild\MonoGame\v3.0\Tools内的全部文件到 .\Depend\2MGFX 文件夹下之后编译使用。
Pls copy all the files inside "C:\Program Files (x86)\MSBuild\MonoGame\v3.0\Tools" to the folder ".\Depend\2MGFX" by yourself first.
